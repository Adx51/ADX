# ADX — Cave à vin (Home Assistant add-on)

Premium AI-powered wine cellar management, running as a Home Assistant add-on
(web UI + API + PostgreSQL/pgvector, all in one container).

Add `https://github.com/Adx51/ADX` as an add-on repository in Home Assistant,
then install **ADX — Cave à vin**. See [DOCS.md](DOCS.md) for full instructions.

Demo login: `demo@adx.wine` / `demo1234`.
